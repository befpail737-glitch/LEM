# LEM Distributor Website

LiTong Electronics is the core first-tier distributor of LEM (LEM), providing the full range of sensor products and technical support services.

## Site Structure

```
/
├── index.html                 # Home page
├── about.html                 # About us
├── contact.html               # Contact page
├── 404.html                   # 404 page
├── sitemap.xml                # Sitemap
├── robots.txt                 # Robots protocol
├── css/
│   ├── style.css              # Main stylesheet
│   ├── variables.css          # CSS variables
│   ├── reset.css              # Style reset
│   ├── layout.css             # Layout styles
│   └── components.css         # Component styles
├── js/
│   └── main.js                # Main JavaScript file
├── svg/
│   ├── logo.svg               # Company Logo
│   └── favicon.svg            # Website icon
├── products/
│   └── index.html             # Products center
├── solutions/
│   └── index.html             # Solutions
├── support/
│   └── index.html             # Technical support
└── news/
    └── index.html             # News center
```

## Technical Specifications

- **Tech Stack**: Native HTML + CSS + JavaScript
- **Responsive Design**: Mobile-first approach
- **SEO Optimized**: Complete Meta tags and structured data
- **Deployment**: Cloudflare Pages
