# LEM Distributor Website

This is the official website for LiTong Electronics, an authorized LEM sensor distributor. The site is built using vanilla HTML, CSS, and JavaScript as specified in the project requirements.

## Project Structure

```
website/
├── index.html (Home page)
├── robots.txt
├── sitemap.xml
├── src/
│   ├── css/
│   │   └── style.css (Main stylesheet)
│   ├── js/
│   │   └── main.js (Main JavaScript)
│   ├── images/
│   └── icons/
├── products/
│   ├── index.html (Product center)
│   ├── hall-sensors/
│   ├── voltage-transducers/
│   ├── high-precision/
│   ├── automotive/
│   └── rogowski-coils/
├── solutions/
│   ├── index.html (Solutions overview)
│   └── bms/ (Battery Management System solution)
├── support/
│   └── index.html (Technical support)
├── news/
│   └── index.html (News center)
├── about/
│   └── index.html (About Us)
└── contact/
    └── index.html (Contact page)
```

## Features

- Fully responsive design for all device sizes
- SEO optimized with proper meta tags and schema markup
- Product filtering and comparison functionality
- Technical support resources section
- Contact forms with validation
- Mobile-friendly navigation

## Technology Stack

- HTML5 for structure
- CSS3 for styling (with modern features like flexbox and grid)
- JavaScript for interactivity (no frameworks used)
- Schema.org markup for rich snippets

## Deployment

This site is designed to be deployed on Cloudflare Pages, as specified in the requirements. It's a static site that can be deployed directly from a Git repository.

## Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
