# LEM Website Testing Checklist

## 1. Cross-Browser and Device Testing

### Desktop Browsers
- [ ] Chrome (latest version)
- [ ] Firefox (latest version)
- [ ] Safari (latest version)
- [ ] Microsoft Edge (latest version)
- [ ] Opera (latest version)

### Mobile Browsers
- [ ] Chrome Mobile (iOS)
- [ ] Safari Mobile (iOS)
- [ ] Chrome Mobile (Android)
- [ ] Samsung Internet (Android)

### Screen Sizes
- [ ] 1920x1080 (desktop)
- [ ] 1366x768 (laptop)
- [ ] 1280x1024 (tablet landscape)
- [ ] 768x1024 (tablet portrait)
- [ ] 375x667 (iPhone/X)
- [ ] 414x896 (iPhone Plus)
- [ ] 360x640 (Android)

## 2. Performance Testing

### Core Web Vitals
- [ ] Largest Contentful Paint (LCP) < 2.5s
- [ ] First Input Delay (FID) < 100ms
- [ ] Cumulative Layout Shift (CLS) < 0.1

### Page Load Times
- [ ] First Contentful Paint (FCP) < 1.5s
- [ ] Time to Interactive (TTI) < 3s

### Speed Tests
- [ ] Google PageSpeed Insights (Desktop) > 90
- [ ] Google PageSpeed Insights (Mobile) > 90
- [ ] GTmetrix score > 90%
- [ ] WebPageTest results (multiple tests)

## 3. SEO Testing

### Technical SEO
- [ ] All pages indexed in Google Search Console
- [ ] XML sitemap submitted and error-free
- [ ] Robots.txt properly configured
- [ ] Canonical tags present and correct
- [ ] Title tags unique and descriptive
- [ ] Meta descriptions unique and compelling
- [ ] Header hierarchy (H1-H6) properly structured
- [ ] Image alt attributes present and descriptive
- [ ] Internal linking working correctly
- [ ] No broken links or 404 errors

### Schema Markup
- [ ] Organization schema valid
- [ ] WebPage schema valid
- [ ] Product schema valid (on product pages)
- [ ] Article schema valid (on news/support articles)
- [ ] Breadcrumb schema valid
- [ ] All schemas pass Google's Rich Results Test

### Keyword Ranking
- [ ] Target keywords ("LEM代理", "LEM代理商") ranking improvement
- [ ] Site visibility in Google Search Console
- [ ] Click-through rates for target pages

## 4. User Experience Testing

### Navigation

- [ ] Internal links open correctly

### Content

- [ ] Images display properly
