# LEM English Website - User Personas

## Overview
This document outlines the primary user personas for the LEM English Website. These personas represent the diverse set of professionals who will engage with the website, from design engineers seeking specific sensor solutions to procurement managers evaluating suppliers for large-scale implementations.

## Persona 1: Design Engineer
- **Name**: Alex Chen
- **Role**: Senior Hardware Design Engineer
- **Company**: Industrial automation manufacturer
- **Age**: 32
- **Location**: Shanghai, China / Berlin, Germany

### Background
Alex has 8 years of experience designing power electronics systems for industrial drives and renewable energy applications. He regularly needs to select current and voltage sensing solutions that meet specific accuracy, isolation, and environmental requirements for new product designs. Alex values technical precision and reliability in sensor solutions.

### Goals & Motivations
- Find the most suitable LEM sensor for specific technical requirements
- Access detailed technical specifications and application notes
- Compare different sensor technologies (Hall effect open-loop vs. closed-loop vs. fluxgate)
- Understand how sensors will perform under specific environmental conditions
- Obtain samples for bench testing and validation

### Pain Points
- Time constraints when evaluating multiple sensor options
- Difficulty finding application-specific guidance
- Need for technical support during integration
- Ensuring component availability throughout product lifecycle
- Balancing performance requirements with cost considerations

### Web Behavior
- Spends significant time reviewing technical documentation
- Downloads datasheets and application notes
- Uses filters to narrow down sensor options by technical parameters
- Searches for specific part numbers or technical challenges
- Values comparison tools and selection guides

### Search Intent
- "High accuracy current sensor for BMS applications"
- "LEM voltage transducer for industrial drives"
- "Difference between open-loop and closed-loop current sensors"
- "LEM current sensor for harsh environment applications"
