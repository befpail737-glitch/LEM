# LEM English Website - Website Structure & Information Architecture

## Overview
This document defines the comprehensive site architecture, navigation structure, user pathways, and information hierarchy for the LEM English Website. The structure is designed to support SEO objectives for "LEM代理" keywords while providing intuitive navigation for global technical professionals and procurement teams.

### URL Structure Strategy
- **Consistent Pattern**: All URLs use lowercase with hyphens as separators
- **Keyword Integration**: Primary keywords integrated where appropriate in URLs
- **Logical Hierarchy**: Clear parent-child relationships in URL structure
- **SEO Friendly**: Clean, descriptive URLs without parameters

5. **Detail Review**: Individual product pages with specifications

#### Journey 2: Solution Discovery

2. **Application Selection**: Electric Vehicle BMS
3. **Solution Review**: Technical diagrams and advantages
4. **BOM Analysis**: Recommended LEM products with links
5. **Related Products**: Discover similar applications
